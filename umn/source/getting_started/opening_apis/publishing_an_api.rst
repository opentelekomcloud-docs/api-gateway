Publishing an API
=================

#. In the navigation pane, choose **API Publishing** > **APIs**.
#. Locate the API created in :ref:`Creating an API <en-us_topic_0080101678>`, and click **Publish**.
#. Select the environment where the API will be published.
#. Click **Publish**.
